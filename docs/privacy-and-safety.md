# Privacy and Safety

GammaRemover is designed as a browser-based cleanup tool for Gamma-exported PDF and PPTX files.

## General safety principles

* Use the tool only for files you own or have permission to modify.
* Keep a copy of the original file.
* Do not process files that contain sensitive information unless you understand your own privacy requirements.
* Do not use cleanup tools to misrepresent authorship or ownership.

## Browser-based cleanup

GammaRemover is positioned as a browser-based cleanup tool. This helps reduce unnecessary file handling compared with upload-based workflows.

Users should still review their own privacy requirements before using any online or browser-based tool.
