# Troubleshooting

This page explains common issues when cleaning Gamma PDF or PPTX exports.

## No watermark was detected

Possible reasons:

* The file was not exported from Gamma.
* The file does not contain supported Gamma branding.
* The branding is stored in an unsupported structure.
* The file was already cleaned.

## The watermark is still visible

Possible reasons:

* The branding was flattened into an image.
* The branding is part of the page background.
* The file uses an unusual export structure.
* The visible object was not detected.

Try the PDF tool here: https://gammaremover.com/en/pdf/

## The PPTX layout changed

PPTX files can contain slide objects, layouts, and master elements. Some files may behave differently depending on how they were exported.

Try the PPTX tool here: https://gammaremover.com/en/pptx/

## The file is too large

Large files may take longer to process and may depend on browser memory.

Try using a desktop browser and close unused tabs.

## The browser ran out of memory

If processing fails:

* Use a desktop browser.
* Close other tabs.
* Try a smaller file.
* Keep the original file.
* Try exporting again from Gamma.

## Mobile browser issues

Mobile browsers may have stricter memory limits. For larger PDF or PPTX files, a desktop browser is recommended.
