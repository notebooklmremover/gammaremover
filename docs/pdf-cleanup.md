# Gamma PDF Cleanup Notes

Gamma PDF exports may contain visible branding, linked badges, footer elements, or positioned objects.

GammaRemover PDF tool: https://gammaremover.com/en/pdf/

## What can usually be cleaned

GammaRemover is designed to clean supported visible branding from standard Gamma PDF exports where the branding is stored as a removable object, link, annotation, or positioned element.

## What may not be cleanable

Some PDF exports may flatten visual content into a single image. If the branding is merged into the page image, cleanup may be limited.

## Recommended use cases

* Cleaning your own Gamma PDF export
* Preparing a PDF for review or presentation
* Removing supported visible branding objects where allowed
* Keeping a cleaner copy of a file you have permission to modify

## Best practice

Always keep the original PDF before processing.
