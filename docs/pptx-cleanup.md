# Gamma PPTX Cleanup Notes

Gamma PowerPoint exports may contain branding as slide objects, shapes, master/layout elements, or linked objects.

GammaRemover PPTX tool: https://gammaremover.com/en/pptx/

## How PPTX cleanup differs from PDF cleanup

PPTX files are editable presentation files. They may contain multiple slide layers, master slides, layout elements, shapes, text boxes, and links.

GammaRemover is designed to remove supported branding elements from standard Gamma-exported PPTX files where those elements can be identified.

## What can usually be cleaned

Supported branding may appear as:

* Shapes
* Text boxes
* Linked objects
* Layout elements
* Slide-master elements

## What may vary

Some exports may use unusual structures. Results may vary depending on how the PPTX file was generated.

## Best practice

Always keep the original PPTX before processing.
