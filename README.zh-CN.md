# GammaRemover

GammaRemover 是一个浏览器端 Gamma PDF 和 PPTX 导出清理工具，用于处理标准 Gamma 导出中可支持的可见品牌元素。

请只处理你自己拥有或有权修改的文件。GammaRemover 不隶属于 Gamma.app，也不代表 Gamma.app。

## 快速入口

* GammaRemover 首页: https://gammaremover.com/zh/
* PDF 清理工具: https://gammaremover.com/zh/pdf/
* PPTX 清理工具: https://gammaremover.com/zh/pptx/

## 支持格式

| 格式 | 支持 | 说明 |
| --- | --- | --- |
| PDF | 是 | 适用于标准 Gamma PDF 导出，结果取决于文件导出结构。 |
| PPTX | 是 | 适用于标准 Gamma PowerPoint 导出中的形状、链接对象或布局元素。 |
| PNG / JPG | 否 | 扁平图片导出目前不支持。 |
| Video | 否 | 不支持视频清理。 |

## 负责使用

Use only for files you own or have permission to modify.

Results may vary depending on how the file was exported. Always keep a copy of your original file.
