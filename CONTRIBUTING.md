# Contributing

Thank you for your interest in improving GammaRemover documentation.

This repository is mainly for project documentation, feedback, troubleshooting notes, and responsible-use guidance.

## What you can contribute

You can submit:

* Documentation improvements
* Troubleshooting notes
* Format-specific cleanup notes
* PDF/PPTX export observations
* Responsible-use improvements
* Broken link fixes
* Accessibility improvements
* Clearer wording for non-technical users

## What not to submit

Please do not submit:

* Unauthorized file-sharing tools
* Tools for stripping protected ownership or licensing information
* Fake official branding
* Malware
* Spam
* Unrelated affiliate pages
* Content that encourages misuse of third-party platforms

## Responsible-use requirement

Any contribution should follow this principle:

Use cleanup tools only for files you own or have permission to modify.

GammaRemover is not affiliated with Gamma.app.
