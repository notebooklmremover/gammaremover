# GammaRemover

GammaRemover는 Gamma에서 내보낸 PDF 및 PPTX 파일을 브라우저에서 정리하는 도구입니다. 지원되는 가시적 브랜드 요소가 분리된 형태일 때 가장 잘 작동합니다.

Use only for files you own or have permission to modify. GammaRemover is not affiliated with, endorsed by, or sponsored by Gamma.app.

## Quick links

* GammaRemover homepage: https://gammaremover.com/ko/
* PDF cleanup tool: https://gammaremover.com/ko/pdf/
* PPTX cleanup tool: https://gammaremover.com/ko/pptx/

## Supported formats

| Format | Supported | Notes |
| --- | --- | --- |
| PDF | Yes | For standard Gamma PDF exports. Results depend on export structure. |
| PPTX | Yes | For standard Gamma PowerPoint exports with removable slide or layout elements. |
| PNG / JPG | No | Flattened image exports are not currently supported. |
| Video | No | Video cleanup is not supported. |

## Responsible use

Results may vary depending on how the file was exported. Always keep a copy of your original file.

Do not use cleanup tools to misrepresent authorship or violate platform terms.
